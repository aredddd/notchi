# Notchi 1.1.4

This release focuses on smoother day-to-day performance, safer agent hook controls, and a few small reliability fixes around the notch experience.

## Performance

Notchi does less background work while keeping Claude and Codex activity fresh.

- Reduces repeated Codex session scanning and batches metadata lookups
- Moves launch-time shell probes and user-agent resolution off the main thread
- Avoids rewriting hook scripts and settings files when nothing changed
- Refreshes idle mode only when the hide-sprite preference changes
- Slows idle sprite bobbing and respects Reduce Motion more consistently

## Agent Hooks

Agent hook controls are safer and easier to reason about.

- Adds per-provider toggles to disable and uninstall Claude and Codex hooks
- Prunes only Notchi-managed Claude hook commands during uninstall
- Logs corrupt hook config files instead of silently leaving stale references
- Keeps hook install failures distinct from not-installed state in settings

## Fixes

- Keeps Codex rollout tailing from dropping a line at window boundaries
- Replaces the global mouse-move monitor with notch tracking-area handling
- Skips notch tracking setup until geometry is ready
- Shows global shortcut modifiers in standard macOS order
- Prevents Cmd-M and Cmd-Option-M from minimizing the notch panel
